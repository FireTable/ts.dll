# ts.dll

## 简介
**通过nodejs调用天使插件,主要参考了aweiu的dm.dll,致敬!**
详情API可以参考目录中的: `Ts帮助文档.chm`  

## 如何使用

```bash
# Clone this repository
git clone https://github.com/FireTable/ts.dll
# Go into the repository
cd ts.dll
# Install dependencies
npm install
# Run
node ./src/index.ts
# Build
npm run build
```
